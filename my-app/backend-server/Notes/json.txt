
A JSON object is a collection of key-value pairs. 
The keys are strings, and the values can be any JSON data type, 
including strings, numbers, booleans, null values, arrays, and other objects.


{
  "name": "John Doe",
  "age": 30,
  "city": "New York",
  "hobbies": [
    "reading",
    "hiking",
    "traveling"
  ],
  "address": {
    "street": "123 Main Street",
    "city": "New York",
    "state": "NY",
    "zip": "10001"
  }
}


This JSON object represents a person named John Doe who lives in New York City.
He has three hobbies: reading, hiking, and traveling.
He also has an address with a street address, city, state, and zip code.
